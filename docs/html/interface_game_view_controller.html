Detailed Description

The actual class that displays the output of the C++ renderer. The class inherits MtkViewDelegate to be able to receive information from the GPU device. Also, GameViewController acts as a bridge of sorts between the C++ engine and the rest of the UI written in Swift and using Apple's frameworks.
